Fake Image Detection using Deep Learning
Published in IJIRT · Volume 10 Issue 11 · ISSN: 2349-6002 · April 2024
Co-authored Paper
Built an image authenticity verification system using CNNs and GANs to combat fake image proliferation in digital journalism. Designed for real-time analysis with a Streamlit user interface, the system classifies images as authentic or manipulated using a hierarchical CNN architecture trained on Kaggle datasets. Research addressed misinformation challenges faced by journalists and content curators.
Problem
55% of surveyed users had encountered fake images; only 45% trusted media image authenticity
Architecture
CNN with sliding window, batch norm, dropout, max pooling — optimized for real-time detection
ML Model
CNNs + GANs for forgery detection; trained on 1200+ image dataset (80/10/10 split)
Outcome
Real-time classification UI; paper published internationally; future scope for video & audio deepfakes
